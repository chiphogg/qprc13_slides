Overview

The basic story is this.

We've got a metal plate, and we stretch it. So there's some amount of strain at every point on this surface. We've measured this strain function at a bunch of points, but we wish we knew it everywhere. We wish we knew this strain surface, if you like. We have some idea about it, but we're not completely certain.

So, first question: how uncertain are we? You see we'll need a sensible way to quantify uncertainty in continuous functions.

Gaussian process machinery:

  • There's some function you wish you knew.
  • Give the machinery some arbitrary number of points where you know the function (it's fine if they're noisy).
  • Ask it for the values of the function at some other set of points
    • The key is: look at those values simultaneously.
